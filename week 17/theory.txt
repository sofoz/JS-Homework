2. Какими методами можно найти подстроку в строке? Приведите пример.
Существует несколько способов поиска подстроки:
1) str.indexOf(substr, pos) - ищет подстроку substr в строке str, начиная с позиции pos, и возвращает позицию, на которой располагается совпадение, либо -1 при отсутствии совпадений.

let str = 'Widget with id';
alert( str.indexOf('id', 2) ); // 12

2)  str.lastIndexOf(substr, position) - возвращает индекс последнего вхождения указанного текста в строку.

let str = 'Индекс последнего вхождения'
alert( str.lastIndexOf('индекс')); //-1

3) search() - ищет строки по заданному значению и возвращает позицию совпадения, может принимать гораздо более мощные значения поиска (регулярные выражения).

let str = 'Я учу JS'
alert(str.search(JS)); //6

6. Как посчитать, сколько секунд осталось до завтра?

let today = new Date();
let tomorrow = new Date(today.getFullYear(), today.getMonth(), today.getDate() + 1);
let sec = (tomorrow - today) / 1000;
alert(sec)

7. Как выделить из строки с денежной суммой (например, *120р.* или *99€*) только цифры, т.е. *120 и 99*?
Функция parseInt() принимает строку в качестве аргумента и возвращает целое число в соответствии с указанным основанием системы счисления. 10 - основание системы счисления числовой строки (чаще всего десятичная)
parseInt('120р.', 10);
parseInt('99€', 10);
Либо можно использовать метод slice
